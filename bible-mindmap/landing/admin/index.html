🔐 GitHub Personal Access Token 설정

어드민 편집은 GitHub API를 통해 저장소의 landing/data/content.json을 직접 커밋합니다. 아래 순서로 Fine-grained PAT를 발급해 붙여넣으세요.

  1. GitHub → Fine-grained tokens 발급 페이지 열기
  2. Repository access: "Only select repositories" → bible-mindmap 선택
  3. Repository permissions: Contents → Read and write
  4. 만료 기간 설정 후 Generate token 클릭
  5. 발급된 토큰(ghp_ 또는 github_pat_ 로 시작) 복사 → 아래 붙여넣기
토큰은 브라우저 localStorage에만 저장되며 어디로도 전송되지 않습니다.
