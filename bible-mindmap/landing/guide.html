01

앱 개요

성경 마인드맵은 성경 구절·인물·장소·시대를 캔버스 위에 노드로 올려놓고, 서로 연결선을 그어 관계를 시각화하는 인터랙티브 도구입니다. 설치 없이 브라우저에서 바로 사용하며, 작업한 내용은 브라우저 로컬 스토리지에 자동 저장됩니다.

🗺️

무한 캔버스

줌·패닝이 자유롭고 노드를 어디든 배치할 수 있습니다.

📜

원문 성경

헬라어 신약(NTGT), 히브리어 구약(WLC), 개역한글(KRV), ESV를 탭으로 전환합니다.

🔤

원문 형태소 분석

단어를 클릭하면 Strong's 번호·형태소·사전 정의·이 책에서의 용례를 팝업으로 확인합니다.

🔗

교차 참조

OpenBible 데이터 기반으로 연관 구절을 자동 추천하고 캔버스에 바로 추가합니다.

저장 위치: 모든 캔버스 데이터는 현재 브라우저의 로컬 스토리지에만 저장됩니다. 다른 기기나 다른 브라우저에서는 보이지 않으므로, 중요한 작업은 JSON 파일로 내보내 백업하세요.
02

캔버스 기본 조작

React Flow 기반의 무한 캔버스입니다. 마우스와 트랙패드 제스처를 모두 지원합니다.

동작방법
화면 이동 (패닝)빈 공간을 드래그 / 트랙패드 두 손가락 스와이프
줌 인/아웃마우스 휠 / 트랙패드 핀치 / 우측 하단 +/− 버튼
전체 보기우측 하단 □ 버튼 (Fit View)
노드 이동노드 헤더 영역 드래그
노드 크기 조절노드 오른쪽 하단 모서리 드래그
노드 선택클릭 (복수 선택: Shift+클릭 또는 드래그 박스)
노드 삭제선택 후 Delete / Backspace
연결선 그리기노드 엣지의 핸들(●)을 다른 노드로 드래그
연결선 삭제연결선 클릭 후 Delete
실행취소 / 재실행Ctrl+Z / Ctrl+Shift+Z
미니맵: 우측 하단의 작은 지도를 클릭·드래그해 캔버스 위치를 빠르게 이동할 수 있습니다.
03

노드 종류

왼쪽 사이드바에서 원하는 노드 타입을 선택해 캔버스에 추가합니다.

📖

구절 노드 (Verse)

성경 구절을 담는 핵심 노드. 책·장·절을 지정하면 번역과 원문을 자동으로 불러옵니다. 원문 탭·교차 참조·렉시콘 팝업이 여기서 작동합니다.

📝

메모 노드 (Note)

자유롭게 텍스트를 입력하는 노드. 주해·설교 아이디어·개인 묵상을 적을 때 사용합니다.

💡

주제 노드 (Topic)

신학적 주제·키워드를 표시하는 노드. 구절들과 연결해 주제 중심 지도를 만들 수 있습니다.

👤

인물 노드 (Person)

성경 인물을 검색하면 Wikidata에서 생몰 연도·설명을 자동 로드합니다. 동시대 인물 버튼으로 관련 인물을 한 번에 추가합니다.

📍

장소 노드 (Place)

성경 지명을 검색해 좌표와 설명을 자동으로 가져옵니다. 연관 인물도 탐색할 수 있습니다.

🕰️

시대 노드 (Period)

족장·출애굽·왕정·포로·귀환·중간·사도 등 성경의 주요 시대를 표시하는 노드입니다.

04

구절 노드 심화

구절 노드는 가장 강력한 기능을 제공합니다. 노드를 선택한 뒤 상단 편집 바에서 모든 기능에 접근할 수 있습니다.

1

구절 추가

좌측 사이드바 → 구절(📖) 선택 → 책·장·절 입력 → 추가. 또는 편집 바의 책·장·절 입력란에 직접 입력합니다.

2

번역 탭 전환

노드 내 탭 버튼으로 개역한글 ESV 원어를 전환합니다. 원어 탭에서는 헬라어 신약(NTGT) 또는 히브리어 구약(WLC)을 표시합니다.

3

원문 탭 — 단어 클릭

원어 탭 활성화 후 단어를 클릭하면 렉시콘 팝업이 열립니다. 팝업은 헤더를 드래그해 위치를 자유롭게 옮길 수 있습니다.

4

교차 참조

편집 바의 🔗 교차 참조 버튼을 클릭하면 OpenBible 투표 수 기반 관련 구절 목록이 나타납니다. + 추가 버튼으로 캔버스에 바로 연결해 추가합니다.

5

병렬 보기 (단어 페어링)

편집 바의 단어 페어링 버튼으로 원문↔번역 단어를 나란히 놓고, 대응하는 단어쌍을 색깔로 표시하는 분석 화면이 열립니다.

05

원문 렉시콘 & 어형 분석 팝업

원문 탭에서 단어를 클릭하면 나타나는 팝업입니다. 본문형(어형) 표시 → 스트롱 번호 → 레마 사전 조회 의 세 단계로 연결됩니다.

📌 본문형(어형) vs. 레마(기본형) — 무엇이 다른가?
구분 헬라어 예시 히브리어 예시 설명
본문형 (칩) δοῦλοι בָּרָא 성경 본문에 실제 등장한 활용형 (복수·격·시제 등 변화 포함)
레마 (팝업) δοῦλος בָּרָא 사전의 표제어 역할을 하는 기본형 (Strong's 번호로 연결)
Strong's 번호 G1401 H1254 레마와 1:1 대응. 팝업 사전 조회의 열쇠

즉, 칩은 "이 단어가 본문에서 어떤 형태로 쓰였는가"를 보여주고,
팝업은 "그 단어 어근의 기본 의미가 무엇인가"를 알려줍니다.

📖

팝업 상단 — 형태소 정보

Strong's 번호 — 클릭 시 BibleHub 원문 사전으로 이동
레마 (l:) — 사전 기본형 (있는 경우)
음역 (tr:) — 라틴 문자 발음 표기
형태소 (g:) — 문법 분석 (격·수·성·태·법·시제 등)
기본뜻 — STEPBible 영어 글로스

📚

팝업 하단 — 사전 정의

헬라어 — openscriptures/strongs 공개 사전 정의 (Public Domain)
히브리어 — BDB(Brown·Driver·Briggs) 상세 정의 + openscriptures 보완
정의 내 H1234·G5678 코드를 클릭하면 관련 단어 BibleHub 페이지로 이동

다중 팝업 & 이동: 여러 단어를 클릭하면 팝업이 각각 독립적으로 열립니다. 헤더를 드래그해 원하는 위치로 이동. X 버튼 또는 Esc 키로 닫습니다.
데이터 출처: STEPBible TAGNT/TAHOT (CC BY 4.0) · openscriptures/strongs (Public Domain) · Bolls.life BDB · BibleHub (외부 링크 참조용)
06

교차 참조 (Cross-Reference)

OpenBible.info의 크라우드소싱 교차 참조 데이터를 기반으로, 선택한 구절과 주제적으로 연관된 다른 구절을 자동 추천합니다.

1

교차 참조 열기

구절 노드 선택 → 상단 편집 바 → 🔗 교차 참조 버튼 클릭. 투표 수 기준 상위 12개 관련 구절이 표시됩니다.

2

캔버스에 추가

목록에서 원하는 구절의 + 추가 버튼을 클릭하면 번역을 자동으로 불러오고, 원본 노드 오른쪽에 새 구절 노드가 생성되며 교차 참조 엣지(점선)로 연결됩니다.

3

패널 닫기

🔗 교차 참조 버튼을 다시 클릭하면 패널이 닫힙니다.

07

병렬 보기 & 단어 페어링

원문과 번역을 나란히 놓고 단어 대응 관계를 색깔로 표시하는 심층 분석 화면입니다. 설교 준비·원어 연구에 유용합니다.

1

열기

구절 노드 선택 → 편집 바 → 단어 페어링 버튼. 원문 열(헬라어/히브리어)과 번역 열이 나란히 열립니다.

2

단어 대응 표시

원문 단어를 클릭하면 파란색으로 선택됩니다. 번역 열에서 대응하는 단어를 클릭하면 같은 색으로 연결됩니다. 최대 8가지 색상 쌍을 지원합니다.

3

원문 단어 렉시콘

병렬 보기 안에서도 원문 단어를 Cmd+클릭(Mac) / Ctrl+클릭(Windows)하면 렉시콘 팝업이 열립니다.

4

저장

💾 저장 후 닫기를 클릭하면 페어링 정보가 노드에 반영됩니다. 취소는 우측 상단 ✕로 닫으면 됩니다.

08

저장 & 불러오기

💾

자동 저장

캔버스가 변경될 때마다 브라우저 로컬 스토리지에 자동으로 저장됩니다. 페이지를 닫았다 다시 열면 이전 작업이 복원됩니다.

📤

JSON 내보내기

우측 패널의 내보내기 버튼으로 현재 캔버스를 JSON 파일로 저장합니다. 다른 기기로 이전하거나 백업할 때 사용합니다.

📥

JSON 불러오기

내보낸 JSON 파일을 불러와 캔버스를 복원합니다. 기존 캔버스는 덮어쓰이므로 주의하세요.

주의: 로컬 스토리지는 브라우저 데이터를 지우면 삭제됩니다. 중요한 작업은 반드시 JSON 파일로 내보내 보관하세요.
